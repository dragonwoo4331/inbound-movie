# Screenshots

This directory contains screenshots of the Inbound Movie application.

## Screenshots to be added:

### Home Page
- Main search interface
- Top rated movies display
- Responsive design on different screen sizes

### Movie Detail Page
- Movie information display
- Favorite toggle functionality
- Loading states

### Authentication Pages
- Login page
- Register page
- Email verification page

### Favorites Page
- User's favorite movies list
- Empty state when no favorites

### API Documentation
- Swagger UI screenshots
- GraphQL Playground screenshots

## How to add screenshots:

1. Take screenshots of the running application
2. Save them in this directory with descriptive names
3. Update the README files to reference these images
4. Example: `![Home Page](./docs/screenshots/home-page.png)`

## File naming convention:

- `home-page.png` - Main homepage
- `movie-detail.png` - Movie detail page
- `login-page.png` - Login page
- `register-page.png` - Registration page
- `favorites-page.png` - Favorites page
- `swagger-api.png` - API documentation
- `graphql-playground.png` - GraphQL interface
- `mobile-responsive.png` - Mobile view
- `loading-states.png` - Loading indicators
